## Address formats

- **eSpace:** Ethereum-style `0x` (42-char hex). Same as Ethereum.
- **Core Space:** CIP-37, e.g. `cfx:aatktb2te25ub7dmyag3p8bbdgr31vrbeackztm2rj` (network prefix + base32).
